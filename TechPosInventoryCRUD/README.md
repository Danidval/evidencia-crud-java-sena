# 📦 TechPos Inventory CRUD

Sistema de Gestión de Inventario desarrollado en **Java** con interfaz gráfica mediante **JOptionPane** y base de datos **MySQL**. 
Implementa las 4 operaciones básicas del CRUD (Create, Read, Update, Delete).

---

## 📋 Requisitos del Sistema

| Requisito | Versión |
|-----------|---------|
| Java JDK | 11 o superior |
| MySQL Server | 8.0 o superior |
| Maven | 3.6+ (opcional) |
| Sistema Operativo | Windows / Linux / macOS |

---
